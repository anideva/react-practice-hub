# React Practice Hub 🚀

A structured workspace tracking my learning journey, core concepts, and micro-projects in React.

## 📁 Repository Structure
- **01-hooks/**: Exploring useState, useEffect, and custom hooks.
- **02-state-management/**: Implementations of Context API and external state tools.
- **03-routing/**: Page navigation and dynamic routes.
- **04-performance/**: Optimization exercises using useMemo and useCallback.

## 🛠️ Tech Stack
- React (Vite)
- JavaScript / TypeScript
- CSS / Tailwind CSS

- ### 🛠️ How to Test This Code
Since this is an isolated component file:
1. Copy the code inside `CounterApp.jsx`.
2. Open an online React environment like [OneCompiler React](https://onecompiler.com).
3. Paste the code into the main application file (overwriting `App.js`) to see it run.
